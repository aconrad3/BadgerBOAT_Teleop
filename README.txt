1) Install Arduino IDE on Jetson Nano: https://www.jetsonhacks.com/2019/10/04/install-arduino-ide-on-jetson-dev-kit/
2) Connect Arduino Uno to Jetson Nano via USB cable
3) Upload .ide file to Arduino Uno from Jetson Nano
4) Open terminal, run .py file
WASD to control, hit any other key to abort movement, q to quit program
