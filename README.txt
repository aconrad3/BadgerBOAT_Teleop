1) Install Arduino IDE on Jetson Nano: https://www.jetsonhacks.com/2019/10/04/install-arduino-ide-on-jetson-dev-kit/
2) Connect Arduino Uno to Jetson Nano via USB cable
3) Upload .ide file to Arduino from Jetson Nano
4) Open terminal, type xhost + 
(Documented here: https://stackoverflow.com/questions/31902846/how-to-fix-error-xlib-error-displayconnectionerror-cant-connect-to-display-0)
5) run .py in terminal as sudo python DecemberBadgerBOAT_Teleop.py



WASD to control
